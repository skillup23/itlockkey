# RDP Wrapper

Для установки и настройки rdp используется Windows 10 pro 64bit со всеми обновлениями по состоянию на декабрь 2022 года.

1. Качаем версию 1.6.2 с гитхаба https://github.com/stascorp/rdpwrap/releases в виде zip архива. Не забываем добавлять в исключения антивируса всю папку.
2. Распаковываем ее по пути `%ProgramFiles%\RDP Wrapper` строго без версий на конце
3. Качаем **Autoupdater из** [https://github.com/asmtron/rdpwrap/blob/master/binary-download.md](https://github.com/asmtron/rdpwrap/blob/master/binary-download.md) и добавляем его в папку `RDP Wrapper`
4. С правами администратора запускаем autoupdate.bat. Сам rdpWrapper через install устанавливать не нужно
5. Проверяем
