# Nextcloud как служба на сервере

1. Установите клиент nextcloud без опции “Запуск при запуске системы” или отключите данный параметр в уже установленном приложении
2. Настройте папки для синхронизации
3. В планировщике заданий создайте новую задачу по скриншотам ниже
4. При выборе пользователя используйте ту же учетную запись, с которой настраивали и устанавливали Nextcloud
5. Перезагрузите сервер
6. Нет необходимости входить в свою учетную запись, она должна синхронизироваться автоматически
