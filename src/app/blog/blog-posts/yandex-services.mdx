# Установка Яндекс Диска на серверные ОС

Официально альтернативные (Windows Server, Windows Embedded, etc) версии Windows не поддерживаются программой Диска.

Возникает ошибка при авторизации в уже установленную программу, просит войти в Яндекс ID и после ввода всех данных перезапускает форму авторизации. И так циклично.

Проблема решается следующим способом:

1. Удаляем ранее установленное приложение Яндекс Диска с сервера.
2. Устанавливаем Яндекс Браузер, входим в нем в учетку Яндекс ID, ставим этот браузер по умолчанию для системы.
3. Переходим в Яндекс диск в браузере, проверяем авторизацию.
4. Скачиваем повторно из Яндекс Браузера приложение [Скачать установочный файл](https://360.yandex.ru/disk/download/)
5. Устанавливаем и тогда оно автоматом подтягивает данные авторизации из Яндекс Браузера.

Большим минусом является максимальный объем загружаемых файлов - 50 ГБ.
