# Git pull

<div className="mb-0">Добавить все файлы</div>

```bash
git add -A
```

<div className="mb-0">Сделать коммит</div>

```bash
git commit -m "комментарий к коммиту”
```

<div className="mb-0">Отправить изменения в github</div>

```bash
git push
```

<div className="mb-0">Cписок веток</div>

```bash
git branch
```

<div className="mb-0">Создание новой ветки</div>

```bash
git branch name
```

<div className="mb-0">Переход к ветке</div>

```bash
git checkout name
```

<div className="mb-0">Слияние веток, находясь в той куда сливаем.</div>

```bash
git checkout name
```

<div className="text-rose-900">
  Пример: находясь в ветке develop - git merge main - ветку main копируем в
  ветку develop
</div>

<div className="mb-0">Сбросить изменения в локальном репозитории</div>

```bash
git reset --hard
```
