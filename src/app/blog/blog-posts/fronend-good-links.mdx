### [https://swiperjs.com/](https://swiperjs.com/) - интеграция слайдеров на сайт

Слайдеры для интеграции в js, react, vie, angular и т.д. Документацию изучить, но можно по [https://swiperjs.com/demos](https://swiperjs.com/demos) посмотреть как работает и вызвать код в песочнице. Так же задаются определенные параметры в функции, их нужно изучить:

```jsx
var swiperPopular = new Swiper('.popular__container', {
  spaceBetween: 32,
  grabCursor: true,
  centeredSlides: true,
  slidesPerView: 'auto',
  loop: true,

  navigation: {
    nextEl: '.swiper-button-next',
    prevEl: '.swiper-button-prev',
  },
});
```

### [https://boxicons.com/](https://boxicons.com/) - иконки на сайт

[https://boxicons.com/usage](https://boxicons.com/usage) - инструкция по подключению

можно вставлять как веб-компонент

```html
<box-icon name="color"></box-icon>
```

так и в шрифт

```html
<i class="bx bx-color"></i>
```

Пример:

```html
<a href="#contact" class="nav__link">
  <i class="bx bx-phone"></i>
  <span>Contact</span>
</a>
```

### [https://scrollrevealjs.org/](https://scrollrevealjs.org/) - анимация элементов при скролле страницы

интегрируем код с гитхаб скачав [scrollreveal.min.js](https://github.com/jlmakes/scrollreveal/blob/master/dist/scrollreveal.min.js)

```jsx
<script src="assets/js/scrollreveal.min.js"></script>
```

подключаем функцию в основном js файле, где указываем стандартные параметры появления, дистанции, задержки. (reset:true - позволяет проигрывать анимацию каждый раз)

```jsx
const sr = ScrollReveal({
  origin: 'top',
  distance: '60px',
  duration: 2500,
  delay: 400,
  reset: true,
});
```

Далее добавляем классы элементов на странице и задаем им разные параметры если требуется

```jsx
sr.reveal(
  `.home__title, .popular__container, .subscribe__container, .footer__container`
);
sr.reveal(`.home__description, .footer__info`, { delay: 500 });
sr.reveal(`.home__search`, { delay: 600 });
sr.reveal(`.home__value`, { delay: 700 });
sr.reveal(`.home__images`, { delay: 800, origin: 'bottom' });
sr.reveal(`.logos__img`, { interval: 100 });
sr.reveal(`.value__images, .contact__content`, { origin: 'left' });
sr.reveal(`.value__content, .contact__images`, { origin: 'right' });
```
