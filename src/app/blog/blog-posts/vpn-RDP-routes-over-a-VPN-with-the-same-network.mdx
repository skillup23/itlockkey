# Маршруты если RDP через VPN с одинаковой сетью сервера и клиента

На клиенте необходимо в cmd

```bash
route add 0.0.0.0 mask 0.0.0.0 192.168.12.1
```

```bash
route add [ip сервера rdp] mask 0.0.0.0 [шлюз впн]
```

Если при **_ipconfig_** шлюз впн не указан, то берем ip адрес VPN и изменяем конечное значение на **_1_**.

К примеру **_192.168.22.11_** - ip полученный от VPN соединения, значит [шлюз впн] ставим **_192.168.22.1_**
