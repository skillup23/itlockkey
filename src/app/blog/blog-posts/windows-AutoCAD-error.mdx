# Автокад ошибки

## Автокад выдает ФАТАЛЬНУЮ ОШИБКУ: Unhandled Access Violation Reading 0x0108 Exception at 9FB9F16Dh

Проблема:
Как запустить Автокад и продукты на его основе без аппаратного ускорения видео.

Причины:
После включения или изменения функций производительности графики в Автокад программа не может запуститься.

Решение:
В продуктах на базе Автокад 2007 или более поздней версии Perfomance Tuner используется для настройки параметров 3D, визуализации и аппаратного ускорения видео в соответствии с возможностями видеокарты. Изменение этих настроек по умолчанию может вызвать проблемы с производительностью или стабильностью приложения, а также может помешать успешному запуску программы, чтобы запустить Автокад в этом случае, запустите программу с базовым программным ускорением видео. Следуй этим шагам:

1. Шелкните правой кнопкой мыши значок продукта Автокад на рабочем столе или В меню "Пуск" и выберите "Свойство".
2. В диалоговом окне "Свойства" перейдите на вкладку "Ярлык".
3. В поле "Объект" отредактируйте содержимое, чтобы в конце был переключатель /nohardware. Например: `«C:\Programm Files\AutoCAD 2015\acad.exe» /nohardware`
4. Нажмите ОК

Дважды щелкните измененный значок, чтобы запустить Автокад. После успешного запуска программы отмените предыдущие изменения настроек (см. справку по продукту для получения информации о настройке параметров производительности).

Через /nohardware запускаться будет только конкретно через этот ярлык, это не удобно да вы заметите визуальные различия. Если есть ПО ASUS Armory Crate - то при его удалении ошибка уходит.
